# EU Gas Storage Analysis

Python + Jupyter toolkit for European natural gas storage analysis.

## Data sources
- **Storage**: GIE AGSI+ API (free, agsi.gie.eu)
- **TTF prices**: EEX Transparency (free, transparency.eex.com)

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env   # add your API keys
jupyter lab
```

## Notebooks
Run in order: 01 → 02 → 03 → 04 → 05 → 06 → 07

Notebook 07 generates a PDF analysis report.
